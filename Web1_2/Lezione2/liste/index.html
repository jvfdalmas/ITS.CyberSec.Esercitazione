Esercizi con le liste

Creare una pagina web per svolgere i seguenti esercizi:

1: Crea una lista non ordinata con almeno 3 elementi.

  • First item
  • Second item
  • Third item

2: Crea una lista ordinata con almeno 3 elementi.

  1. First item
  2. Second item
  3. Third item

3: Crea una lista non ordinata con una lista ordinata annidata al suo interno.

  • First item
    1. first first item
    2. first second item
    3. first third item
  • Second item
    1. second first item
    2. second second item
    3. second third item
  • Third Item
    1. third first item
    2. third second item
    3. third third item

4: Crea una lista ordinata utilizzando diversi tipi di numerazione o marcatori (es. numeri, lettere, romani).

  1. First item
  2. Second item
  3. Third item
  4. Fourth item

5: Crea una lista ordinata che inizi da un numero diverso da 1.

  1. Not First item
  2. Not Second item
  3. Not Third item
